# 🖥️ Descripción del Proyecto
EcommerceFront es la capa frontend desarrollada en Angular que consume los siguientes microservicios backend:

Microservicio de Clientes
Microservicio de Pedidos (Orders)
Microservicio de Productos
Este proyecto proporciona la interfaz de usuario para una plataforma de comercio electrónico completa, configurada para ejecutarse en el puerto 4200.

# 💻 🔧 Requisitos Previos
Node.js (versión 16.x o superior)
Angular CLI
Git
